## 🌟 Overview
This is a high-performance **Scientific Calculator** desktop application developed using **C#** and the **.NET Framework**. The project is designed to bridge the gap between complex mathematical logic and a professional, user-friendly interface. It features a high-contrast dark aesthetic, optimized for clarity and precision.

## 🚀 Key Features
### 1. Advanced Mathematical Engine
* **Trigonometric Suite:** $sin, cos, tan$ and their inverses ($sin^{-1}, cos^{-1}, tan^{-1}$).
* **Logarithmic Functions:** Support for $log$, natural log ($ln$), $log_2 x$.
* **Power & Roots:** Calculate $x^2, x^3, x^n$, square roots ($\sqrt{x}$), and cube roots ($\sqrt[3]{x}$).
* **Discrete Math:** Built-in **GCD** (Greatest Common Divisor) algorithm and Factorials ($n!$).

### 2. Professional UI/UX Design
* **High-Contrast Theme:** Dark mode background with distinct color-coded buttons.
* **Numeric Systems:** Quick switching for **Binary (BIN)** and **Hexadecimal (HEX)** values.
